# Nebula Shell

Nebula shell is a work in progress desktop environement made for windows. Our goal is to replace windows explorer with a much more sophisticated and visually pleasing experience.

## Features:

- Keyboard first navigation (similar to what you might see in hyprland on linux).
- Auto tilling
- Seemless multi-desktop experience
- Custom File explorer
- Custom Terminal:
    - Includes auto completion from history
    - the ability to use cmdlets as well as powershell functions in the same session
- Game mode (reduces shell effects to not interfere with youir performance while gaming)
- Custom wallpaper engine
- Clipboard history (accessible through Win + V)
- Apps Quick access:
    - Quickly bring to and out of focus the following apps (can be customized):
        - Discord (Win + D)
        - Spotify (Win + M)
        - Github Desktop (Win + G)

- And most importantly a design language unique to the shell.


## Disclaimer:

The shell is still in an experimental state and may not be suitable to everyone.



*This project was heavly inspired by hyprland & caelestiashell (and while these are for arch, Nebula is focused on windows)*
